# Pretabs-Kitty

An interactive setup wizard for [Kitty](https://sw.kovidgoyal.net/kitty/) that lets you configure any number of named tabs, pick which one gets focus on launch, and either wire the result into your `.zshrc` or just grab the script path.

---

## ✨ Features

- **Interactive wizard** — no manual script editing required.
- **Custom tab count & names** — as many tabs as you need, each with the title you choose.
- **Focus control** — pick which tab is active when the script runs.
- **Flexible `.zshrc` integration** — auto-launch on every session, register as a named command, or just keep the script for manual use or portability.
- **Self-contained output** — the generated tab script is placed right next to `pretabs-kitty.sh`.

---

## 📥 Prerequisites

Install [Kitty](https://sw.kovidgoyal.net/kitty/):

```bash
# Debian/Ubuntu
sudo apt-get install kitty
```

Enable remote control in `~/.config/kitty/kitty.conf`:

```
allow_remote_control yes
```

## 🚀 Quick start

```bash
git clone https://github.com/serber1990/Pretabs-Kitty.git
cd Pretabs-Kitty
chmod +x pretabs-kitty.sh
./pretabs-kitty.sh
```

The wizard will ask you four questions:

| Step | Question |
|------|----------|
| 1 | How many tabs do you want? |
| 2 | A name for each tab |
| 3 | Which tab should have focus after launch |
| 4 | Name for the generated script *(default: `kitty-tabs.sh`)* |
| 5 | **1** Auto-launch on every zsh session · **2** Add as a named command · **3** Just save the script |

Then it asks what to do with the generated script — three options:

## 📋 Example session

```
╔══════════════════════════════════════╗
║       Pretabs-Kitty  •  Setup        ║
╚══════════════════════════════════════╝

How many tabs do you want? 3

Name each tab:
  Tab 1 name: Dev
  Tab 2 name: Logs
  Tab 3 name: Git

Which tab should have focus after launch?
  1) Dev
  2) Logs
  3) Git
Tab number [1-3]: 1

Generated script name [kitty-tabs.sh]: work-tabs.sh

  ✔ Script created: /path/to/Pretabs-Kitty/work-tabs.sh

What would you like to do with the generated script?
  1) Add to ~/.zshrc — auto-launch on every new zsh session
  2) Add to ~/.zshrc — as a named command you call manually
  3) Just save it — I'll decide what to do with it
Choose [1/2/3]: 1

  ✔ Added to ~/.zshrc
  The tab layout will launch automatically on every new zsh session.
```

The generated `work-tabs.sh` will look like:

```bash
#!/usr/bin/env bash
# Generated by pretabs-kitty.sh
# Tabs : Dev Logs Git
# Focus: Dev

kitty @ set-tab-title 'Dev'
kitty @ launch --type=tab --tab-title "Logs" --keep-focus
kitty @ launch --type=tab --tab-title "Git" --keep-focus
kitty @ focus-tab --match 'title:Dev'
```

---

## 📝 How the generated script works

| Command | Effect |
|---------|--------|
| `kitty @ set-tab-title 'Name'` | Renames the already-open first tab |
| `kitty @ launch --type=tab --tab-title "Name" --keep-focus` | Opens a new tab without stealing focus |
| `kitty @ focus-tab --match 'title:Name'` | Moves focus to the chosen tab |
